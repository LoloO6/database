Database Labs Repository

This repository contains structured lab exercises and database design implementations completed as part of my Database coursework. 
It demonstrates practical understanding of relational database concepts,transactions, data export and import, queries, normalization, and database design.

Objectives

-Understand core Database Management System (DBMS) concepts
-Design and implement relational database schemas
-Queries
-Apply normalization techniques
-Work with transactions,tables, constraints, joins and etc.

Technologies Used

-PostgreSQL
